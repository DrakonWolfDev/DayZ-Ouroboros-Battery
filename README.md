# DayZ Ouroboros Battery

Part of the **DayZ Admin Tools** suite.

The Ouroboros Battery introduces two high-capacity battery variants designed for admin use and extended gameplay scenarios.

---

## 🔋 Included Variants

### Ouroboros Admin Battery
A true admin power source.

- Effectively unlimited energy capacity  
- Resistant to physical degradation  
- Designed for testing, events, and server administration  
- Ideal for torches, NVGs, and other 9V-powered devices  

This variant is intended as an admin utility. If provided to regular players, balance / impact is the responsibility of the server owner.

---

### Ouroboros Battery (Player Variant)

A high-capacity battery designed for extended field use.

- Significantly increased energy storage compared to vanilla  
- Standard durability behaviour  
- Can still be damaged and eventually ruined  

This version maintains survival mechanics while reducing frequent battery replacement.

---

## 🧩 Design Philosophy

- Modular and additive (no vanilla overrides)
- Clean inheritance structure
- Compatible with standard 9V-powered devices
- Designed to function independently or as part of the wider Admin Tools suite

---

## 🔄 Compatibility

- Tested on DayZ 1.28  
- Tested on DayZ Expansion
